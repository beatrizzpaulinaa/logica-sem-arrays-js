# logica-sem-arrays-js# 💻 Exercícios de Vetores em JavaScript (Sem Métodos Nativos)

Este repositório contém a resolução de quatro exercícios práticos de lógica de programação focados na manipulação de vetores (arrays) em JavaScript. 

O objetivo principal desta atividade foi desenvolver a **lógica pura**. Por isso, os algoritmos foram construídos **sem a utilização de funções nativas de arrays do JavaScript** (como `.push()`, `.includes()`, `.filter()`, etc.), manipulando os índices diretamente através de laços de repetição (`for`).

---

## 📝 Exercícios Resolvidos

1. **Soma de Números Pares**
   - O programa lê 10 números inteiros digitados pelo usuário, armazena-os em um vetor e, em seguida, calcula e exibe a soma exclusivamente dos números pares contidos nele.

2. **Geração de Vetor Aleatório**
   - Preenche automaticamente um vetor numérico com 7 números gerados de forma aleatória pelo computador e exibe os valores gerados na tela.

3. **Contador de Ocorrências**
   - Solicita 10 números inteiros ao usuário e os armazena. Depois, pede que o usuário digite um número específico e verifica quantas vezes esse número aparece dentro do vetor.

4. **Remoção de Duplicatas**
   - Lê 10 números informados pelo usuário. A partir desse vetor original, o algoritmo constrói um novo vetor adicionando apenas os valores únicos (ignorando as repetições). Ao final, exibe o vetor original e o novo vetor sem duplicatas.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** JavaScript (Vanilla)
* **Entrada/Saída de dados:** Funções `prompt()` e `alert()` do navegador.

---

## 🚀 Como executar os códigos

Como os programas utilizam as funções `prompt()` e `alert()`, eles precisam ser executados em um ambiente de navegador. Você pode testá-los de duas formas:

**Opção 1: Arquivo HTML (Recomendado)**
1. Crie um arquivo `.html` (ex: `index.html`).
2. Adicione a tag `<script src="nome-do-arquivo.js"></script>` no corpo do HTML.
3. Abra o arquivo `.html` em qualquer navegador (Chrome, Firefox, Edge, etc.).
